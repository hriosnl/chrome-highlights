# Highlights

Highlight text on webpages. Saves highlights locally with notes.

## Install (unpacked)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder

## Usage

- Select text on any page and press **⌃⌘H** (Mac) or **Alt+H** (Windows/Linux) to highlight it.
- Click the extension icon to view highlights for the current page, pick a color, and copy all highlights.
- Open **All highlights** from the popup to browse your full library across pages.
- Highlights and notes are stored locally in Chrome storage.

## Development

No build step. Edit files and reload the extension in `chrome://extensions`.
